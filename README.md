# MNIST classification (Planet Labs Project) #

### Requirement ###

- Python >=3.4
  - TensorFlow 0.11.0
- Node >=6.9


### How to run ###

    $ pip install -r requirements.txt
    $ git clone https://github.com/AndersonChang/Planet_Labs
    

### Steps Tutorial ###

- Step 1  
	- Open main.py in Sublime or PyCharm IDE
	- Run main.py
